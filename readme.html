Vuxel-Docs ​

The website documentation is built using VitePress and provides comprehensive guides and references for using Vuxel.

Getting Started ​

  • Introduction: Learn about Vuxel and its core concepts in the Introduction section.
  • Installation: Follow the steps to set up Vuxel in your Roblox project in the Installation section.
  • Creating an Application: Build your first Vuxel application by following the guide in the Creating an Application section.
  • Conclusion: Wrap up your learning with some final tips in the Conclusion section.

Examples ​

Explore practical examples to see Vuxel in action:

  • A Simple Button: Learn how to create a simple button in the A Simple Button example.
  • Counter Application: Build a counter application using reactive state and event handling in the Counter Application example.

Dive deeper into Vuxel's API and advanced features:

  • References: Manage references effectively in the References section.
  • Event Handling: Handle events in your UI components in the Event Handling section.

  • Utility Functions: Explore utility functions provided by Vuxel in the Utility Functions section.
  • Animations: Create smooth animations in your UI components in the Animations section.
